# Ivan-silantev Tap

## How do I install these formulae?

`brew install ivan-silantev/tap/<formula>`

Or `brew tap ivan-silantev/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ivan-silantev/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
